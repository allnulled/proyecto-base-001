# proyecto-base-001

Punto de partida para el desarrollo de aplicaciones basadas en vue2 + win7 + socket.io. Opcionalmente nw.js también.

## Instalación

```sh
git clone https://github.com/allnulled/proyecto-base-001.git .
npm install
```

## Uso

Con estos 2 comandos, en consolas diferentes, automatizas la compilación al haber cambios:

```sh
npm run dev
npm run refresh
```

Con este otro comando, compilas:

```sh
npm run build
```

Con este otro comando, arrancas la aplicación con `nw.js`:

```sh
npm start
```
